# **App Name**: Vibe Trail Authentication Suite

## Core Features:

- Login View: Login View with animated transitions and input fields. Includes avatar preview and "Forgot password?" link.
- Sign Up View: Sign Up View with fields for Name, Email, Password, and Confirm Password, and animated transitions.
- Interactive Form Elements: Form elements feature micro-interactions like hover effects and input validation animations.
- Responsive Design: Implement responsive layout that adapts to desktop, tablet, and mobile screens. Stacks layout vertically on smaller screens.
- Page Load Animations: Animate form container on page load (e.g., slide-in).  Use CSS keyframes for background graphics.

## Style Guidelines:

- Primary gradient: Linear gradient from #D4145A to #FBB03B to evoke energy and excitement.  Represents a sunrise.
- Background color: Light gray #F9F9F9 for a clean, modern look. This nearly white background serves to focus user attention.
- Accent Neon: #C770F0 to highlight interactive elements and create a futuristic vibe. Balances vivid energy with calm style.
- Use 'Poppins' (sans-serif) for a modern and clean look across all text elements. Note: currently only Google Fonts are supported.
- Utilize minimalist line-style icons for input fields and other UI elements for a sleek, unobtrusive design.
- Split-screen layout with a graphic/marketing section on one side and the form on the other.
- Add animated transitions when switching between Login and Sign Up views. Use micro-interactions on form elements like input fields and buttons.
